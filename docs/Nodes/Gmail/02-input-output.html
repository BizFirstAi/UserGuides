Input & Output

Output ports and data schemas for all five Gmail node operations.

Output Ports

PortTriggerDescription
successGmail API returned 2xxOperation completed successfully. All response data is available in the output object.
errorAPI error or network failureThe Gmail API returned an error, the OAuth2 token was invalid/expired, or a network failure occurred.

Send Operation Output

FieldTypeDescription
message_idstringThe Gmail message ID assigned to the sent email.
thread_idstringThe thread ID the message belongs to. For new emails, this is a new thread ID. For replies, this matches the existing thread.
label_idsarrayLabels applied to the sent message (typically ["SENT"]).

List Operation Output

FieldTypeDescription
messagesarrayArray of message summary objects, each containing id, thread_id, and snippet.
result_size_estimateintegerEstimated total number of messages matching the query (may differ from actual count).
next_page_tokenstringToken to retrieve the next page of results. Empty when all results have been returned.

Get Operation Output

FieldTypeDescription
message_idstringGmail message ID.
thread_idstringThread this message belongs to.
label_idsarrayLabels currently applied to the message.
fromstringSender email address and display name.
tostringPrimary recipient(s).
subjectstringEmail subject line.
datestringMessage date in RFC 2822 format.
body_textstringDecoded plain text body of the message.
body_htmlstringDecoded HTML body of the message, if present.
attachmentsarrayArray of attachment metadata objects. Each contains attachment_id, filename, mime_type, and size. Data is included when fetch_attachments: true.
snippetstringShort text snippet from the message body, as shown in Gmail's message list.
