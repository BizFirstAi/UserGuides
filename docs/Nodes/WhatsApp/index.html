WhatsApp

Send text, media, and pre-approved template messages via the WhatsApp Business API from within any workflow.

What this node does: The WhatsApp node integrates with the WhatsApp Business API (Meta Cloud API or on-premise) to send messages to customers and staff at scale. Supports freeform text messages within 24-hour customer service windows, pre-approved business templates for outbound notifications, and media messages (images, documents, audio). All phone numbers and message content support BizFirst expressions.

Supported Operations

OperationDescription
sendSend a freeform text or media message. Only valid within the 24-hour customer service window (after customer initiates contact).
sendTemplateSend a pre-approved message template. Can be sent outside the 24-hour window. Required for outbound business-initiated messages.
getMediaDownload a media file (image, document, audio) received from a customer by its media ID.
getStatusRetrieve the delivery status of a previously sent message by its message ID.

Key Capabilities

  • Send text, image, document, and audio messages to any WhatsApp number worldwide
  • Business-approved template messaging for notifications outside the 24-hour window
  • Dynamic template variable injection — order numbers, names, amounts from workflow context
  • Media message support — attach images, PDFs, or audio clips to messages
  • Message delivery status tracking via the getStatus operation
  • Incoming media retrieval for document processing workflows
  • Dual output ports: success and error

Business Benefits

High-Reach Customer Communication

WhatsApp has over 2 billion active users with open rates exceeding 90%. Reach customers where they already are with time-sensitive notifications, confirmations, and support responses directly in their WhatsApp inbox.

Template-Driven Compliance

Meta's pre-approved template system ensures all outbound business-initiated messages comply with WhatsApp's Business Policy. Workflow designers select from approved templates and inject dynamic variables — no ad-hoc messaging outside policy.

Multimedia Delivery

Send invoices as PDFs, product images alongside confirmations, or voice notes for accessibility-focused communications — all from within the workflow without additional file delivery infrastructure.

Use Cases

OTP Delivery

Send one-time passwords for 2FA directly to customers' WhatsApp. Higher delivery rates and faster reads than SMS in many markets.

Appointment Reminders

24 hours before a scheduled appointment, send a personalised WhatsApp reminder with the appointment time, location, and a rescheduling link.

Order Status Updates

At each fulfilment milestone (confirmed, shipped, out for delivery, delivered), send a WhatsApp template message with the current order status and tracking link.

Payment Confirmations

Immediately after a payment is processed, send a WhatsApp receipt with the amount, transaction reference, and support contact.

Field Agent Communication

Send work orders, job details, and document attachments to field agents on WhatsApp. Retrieve photos and documents they send back for workflow processing.
