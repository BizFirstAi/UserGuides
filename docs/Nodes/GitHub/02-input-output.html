Input & Output

Output ports and data schemas for all GitHub node operations.

Output Ports

PortDescription
successGitHub API returned 200/201. Operation completed.
errorGitHub API error — invalid credential, repo not found, missing labels, rate limit exceeded, or network failure.

createIssue / getIssue Output

FieldTypeDescription
issue_numberintegerThe issue number in the repository.
html_urlstringDirect URL to the issue on GitHub.
statestringCurrent state: open or closed.
titlestringIssue title.
created_atstringISO 8601 creation timestamp.
labelsarrayArray of label objects applied to the issue.
assigneesarrayArray of assigned user objects.
node_idstringGraphQL node ID for the issue.

createPullRequest Output

FieldTypeDescription
pr_numberintegerThe pull request number.
html_urlstringDirect URL to the PR on GitHub.
statestringopen or closed.
merge_commit_shastringSHA of the merge commit (populated after merging).
created_atstringISO 8601 creation timestamp.
draftbooleanWhether the PR is a draft.

listIssues Output

FieldTypeDescription
issuesarrayArray of issue summary objects. Each contains number, title, state, html_url, created_at, and labels.
total_countintegerTotal number of issues returned in this response.
