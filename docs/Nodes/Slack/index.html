Slack

Full Slack API integration — post messages, manage channels, upload files, and handle human-in-the-loop approvals directly within workflows.

What this node does: The Slack node provides comprehensive integration with the Slack API. Post messages with rich Block Kit layouts, send interactive approval requests that suspend the workflow until a Slack user responds (HIL pattern), manage channels, upload files, and interact with Slack users — all driven by workflow context.

Supported Resources & Operations

Key Capabilities

  • Rich Block Kit message layouts — sections, dividers, images, buttons, select menus, and more
  • sendAndWait — suspend the workflow mid-execution and resume based on a Slack button click, implementing human-in-the-loop approval patterns entirely within Slack
  • Thread-based replies using thread_ts — keep conversations organised
  • File uploads — share workflow-generated reports, images, and documents in channels
  • Channel lifecycle management — create, rename, archive, and invite to channels programmatically
  • User profile lookup for dynamic mention and routing in messages

Business Benefits

DevOps and Deployment Notifications

Post deployment success/failure messages to engineering channels with colour-coded attachments, commit details, and links to logs. Keep the entire team informed without manual status updates.

Approval Requests in Slack

Replace email-based approvals with interactive Slack messages. Send a formatted approval request with Approve and Reject buttons. The workflow suspends until a designated approver responds — then automatically routes based on their choice.

Customer Support Ticket Alerts

When a high-priority support ticket is created, instantly notify the relevant support channel with ticket details, customer tier, and a direct link to the helpdesk. Assign the ticket from within Slack using interactive buttons.

Weekly Business Reports

Post formatted weekly KPI summaries directly to management channels every Monday morning. Include charts as file uploads, key metrics as Block Kit sections, and comparison to previous period.

Use Cases

CI/CD Pipeline Status

Post build results, test outcomes, and deployment status messages to the #deployments channel. Use Block Kit with coloured status indicators and buttons linking to the CI system.

IT Incident Broadcast

When a monitoring alert fires, post an incident notification to #incidents and #it-ops. Include severity, affected service, start time, and an Acknowledge button for the on-call engineer.

Team Onboarding

When a new employee account is created, send a personalised DM with their onboarding checklist, links to key resources, and a welcome message from their manager.

Purchase Approval

Route purchase requests through a Slack-based approval flow. Present the request details to the approving manager with Approve/Reject/Request-More-Info buttons. Workflow continues or escalates based on the response.

Automated Sales Report

At end of business day, post a formatted sales summary to the #sales channel with deals closed, pipeline movements, and a target vs. actual comparison — pulling data from CRM via upstream HTTP Request nodes.
