Instagram Node

Connect BizFirstAI workflows to the Instagram Graph API. Publish media, moderate comments, query user profiles, search hashtags, send direct messages, and process real-time webhook events — all from a single authenticated node.

Overview

The Instagram node integrates with the Meta Graph API v18.0+ using a long-lived user or page access token. It supports Instagram Business and Creator accounts. All operations are authenticated via an access token stored in the BizFirst Credentials Manager — no raw secrets are stored in workflow configuration.

Business or Creator account required: The Instagram Graph API is available only to Instagram Professional accounts (Business or Creator) connected to a Facebook Page. Personal Instagram accounts cannot use the Graph API.

Authentication

FieldRequiredDescription
accessTokenRequiredLong-lived user or page access token (60-day expiry). Store in BizFirst Credentials Manager. Obtain via auth/getToken, refresh via auth/refreshToken.
igUserIdRequiredInstagram Business/Creator account ID — a numeric string (e.g. 17841400000000000). Retrieve programmatically via page/get.
apiVersionOptionalGraph API version, e.g. v18.0. Defaults to v18.0.
Token expiry: Long-lived tokens expire after 60 days. Schedule an auth/refreshToken node on a 50-day cron to maintain continuous access. The refresh resets the 60-day window.

Operations

Media

media/get

Retrieve a single media object by ID including caption, type, URL, engagement counts, and permalink.

media/getMany

List the media feed for a user with cursor-based pagination and field selection.

media/publish

Publish a media container (step 2 of 2). Pass the creationId from media/createContainer.

media/createContainer

Create a media container (step 1 of 2). Specify image/video URL, caption, location, and user tags.

Comment

comment/get

Retrieve a single comment by ID including text, author, timestamp, and like count.

comment/getMany

List all comments on a media object with cursor-based pagination.

comment/create

Post a comment on a media object owned by your account.

comment/reply

Send a private Instagram Direct reply to a public comment.

comment/delete

Permanently delete a comment from your own media.

comment/hide

Hide a comment from public view without deleting it.

comment/unhide

Restore a previously hidden comment to public visibility.

User

user/get

Retrieve Instagram Business account profile data — bio, follower count, media count, website, profile picture URL.

user/getMedia

List media objects for a specific Instagram user with pagination and field selection.

Hashtag

hashtag/get

Resolve a hashtag name to its stable Graph API ID for use in media search operations.

hashtag/getRecentMedia

Return recent public posts tagged with a hashtag. Up to 10 results per call.

hashtag/getTopMedia

Return algorithmically top-performing posts for a hashtag. Up to 10 results per call.

Messaging

messaging/send

Send a direct message to a user within the 24-hour messaging window.

Page

page/get

Retrieve the Instagram Business account ID linked to a Facebook Page.

Auth

auth/getToken

Exchange a short-lived OAuth code for a long-lived 60-day access token.

auth/refreshToken

Refresh a long-lived token, resetting its 60-day expiry window.

auth/revokeToken

Revoke an access token and disconnect the user from your app.

Trigger

trigger/webhook

Receive and validate Instagram webhook events with HMAC-SHA256 signature verification.
