Node Reference

Complete reference for all BizFirst workflow nodes — triggers, actions, logic, transforms, integrations, and more.

Trigger Nodes

Webhook Trigger Soon

Start a workflow from an inbound HTTP request. Supports GET, POST, authentication headers, and payload mapping.

HTTPInboundREST
Schedule Trigger Soon

Run a workflow on a cron schedule or fixed interval. Supports timezone-aware scheduling and catchup policies.

CronRecurringTimer
Event Trigger Soon

Listen for domain events published to the BizFirst event bus and start a workflow when a matching event arrives.

Event BusAsync
Manual Trigger Soon

Trigger a workflow execution manually from the UI or via the API. Useful for testing and on-demand processes.

ManualAPI

Action Nodes

HTTP Request Soon

Make an outbound HTTP call to any REST or GraphQL API. Supports all verbs, headers, authentication, and response mapping.

HTTPRESTGraphQL
Database Query Soon

Execute SQL queries or NoSQL operations against configured data sources. Returns results into workflow variables.

SQLNoSQLData
Send Email Soon

Send transactional or templated emails via SMTP or configured email providers. Supports dynamic recipients and body templates.

EmailSMTPTemplate
Set Variable Soon

Write one or more values into workflow memory variables using BizFirst expressions. Values are available to all downstream nodes.

VariableMemory

Logic Nodes

Condition Soon

Branch the workflow based on a boolean expression. Routes to the true or false path depending on the evaluated result.

BranchBoolean
Switch Soon

Route the workflow to one of multiple named paths based on a value match. Supports default fallback routing.

Multi-branchRouting
Loop Soon

Iterate over a collection of items, executing a sub-workflow for each item. Supports parallel and sequential execution modes.

IterationCollectionParallel
Wait / Delay Soon

Suspend workflow execution until a fixed time, a duration has elapsed, or an external resume signal is received.

SuspendTimerResume

Transform Nodes

Transform Data Soon

Reshape, rename, and compute fields on the workflow data payload using expressions, mapping rules, and computed fields.

MappingReshapeCompute
Filter Soon

Filter a collection of items using expression-based predicates. Passes only matching items to the next node.

FilterPredicateCollection
Aggregate Soon

Summarise a collection — count, sum, average, min, max, group-by. Results are written into workflow variables.

SumGroup ByMetrics
Merge Soon

Combine the output of multiple parallel branches into a single data object before continuing the workflow.

ParallelJoin

Integration Nodes

Slack Soon

Send messages, post to channels, and handle interactive button responses via Slack. Supports HIL continuation.

MessagingHILSlack
WhatsApp Soon

Send WhatsApp messages and collect user replies as HIL continuation inputs using the WhatsApp Business API.

MessagingHILWhatsApp
Microsoft Teams Soon

Post adaptive cards and collect approval or input responses from Microsoft Teams users mid-workflow.

MessagingHILTeams
SMS Soon

Send SMS messages and handle reply-based HIL continuation via Twilio, AWS SNS, or a custom SMS gateway.

SMSHILTwilio

Human-in-the-Loop Nodes

Approve / Reject Soon

Suspend the workflow and send an approval request to one or more assignees. Continues on approval, routes to rejection path on decline.

ApprovalSuspendRouting
Form Input Soon

Present a dynamic form to a user, suspend the workflow, and resume with the submitted form data mapped into workflow variables.

FormInputSuspend
Review & Correct Soon

Show a human reviewer the current workflow data for inspection. Allows inline corrections before the workflow continues.

ReviewCorrectionAudit

AI Nodes

LLM Call Soon

Send a prompt to a configured large language model (OpenAI, Anthropic, Azure OpenAI) and store the response in a workflow variable.

OpenAIClaudeLLM
Classify Soon

Use an AI model to classify incoming text into predefined categories. Routes the workflow based on the classification result.

ClassificationNLPRouting
Summarise / Extract Soon

Summarise a document or extract structured data fields from unstructured text using a configured AI model.

ExtractionSummaryNLP

Scripting Nodes

JavaScript Node Soon

Execute sandboxed JavaScript (Jint) inline within the workflow. Full access to workflow input, variables, and context.

JavaScriptJintSandbox
C# Script Node Soon

Execute trusted C# scripts via Roslyn within the workflow. Available to Trusted execution level only. Full .NET access.

C#RoslynTrusted
