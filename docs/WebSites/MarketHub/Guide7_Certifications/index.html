Certifications

MarketHub certifications are the trust ladder for packages — a three-level system from automatic community clearance through human-reviewed Certified status to the Official endorsement reserved for BizFirstGO-authored packages.

Why Certification Matters

Certification is the primary trust signal on MarketHub — more than install count or average rating, because installs and reviews can be gamed while certification requires a structured evidence-based review. For enterprise users with governance requirements, filtering by Certified-and-above is often the starting point for any package evaluation.

Certification Roadmap for Publishers

1

Publish Package (Community granted)

Every approved package automatically receives Community certification. The security scan must pass and the manifest must be complete.

2

Meet Certified Criteria

Write comprehensive documentation, achieve 80%+ test coverage, resolve all open critical issues, and maintain a clean security scan.

3

Apply for Certified

Submit a certification application from the publisher workspace with test report and documentation URL as evidence.

4

Certified Granted (3–5 business days)

Admin reviewer evaluates the criteria and evidence. On approval: Certified badge applied, 100 reputation points credited, search ranking improves.

Official is not available to community publishers

The Official level requires a formal partnership with BizFirstGO. Community publishers can achieve Certified — the highest certification available through the standard application process.
