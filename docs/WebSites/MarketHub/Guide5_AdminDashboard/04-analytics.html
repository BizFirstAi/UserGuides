Analytics

The admin analytics page provides marketplace-wide visibility — install trends, category health, top packages, publisher leaderboard movements, and at-risk publisher identification.

Analytics Dashboard Sections

Install Trends

Daily/weekly/monthly install counts across the entire marketplace with date range selector.

Category Health

Installs per category, avg review score per category, new package counts, and growth rate.

Top 10 Packages

Most-installed packages in the selected date range with install count and rating.

Publisher Leaderboard

Ranking changes, tier distribution across the publisher base, new publishers this period.

Search Terms

Top search queries with click-through rates — identify demand for packages that don't yet exist.

At-Risk Publishers

Publishers with declining install counts, increasing issue counts, or no recent activity.

At-Risk Publisher Criteria

The at-risk publisher view flags publishers whose recent metrics suggest they may be abandoning their packages or declining in quality:

  • Install decline — Month-over-month install count down 30%+ for 2 consecutive months
  • Issue backlog — 5 or more open critical issues unresolved for 30+ days
  • No recent activity — No new version, forum post, or issue response in 90 days
  • Rating decline — Average rating dropped 0.5+ stars in the last 60 days

At-risk publishers can be contacted by admins through the issue management section. They do not appear at-risk on the public marketplace — the at-risk view is internal only.
