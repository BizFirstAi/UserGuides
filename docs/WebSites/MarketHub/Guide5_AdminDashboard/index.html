MarketHub

Admin Dashboard

The MarketHub admin portal is the governance and management center — where marketplace administrators review packages, moderate content, configure rules, monitor analytics, and manage the publisher reward system.

Admin access required

The admin portal requires an account with the MarketHub.Admin role. Publisher and general user accounts cannot access the admin portal. Contact your platform administrator to request admin access.

Admin Dashboard Sections

Package Management

Review submitted packages — approve, reject, or delist. Manage the full package lifecycle.

Review Moderation

Moderate user reviews — review flagged content, remove policy violations, add moderator notes.

Marketplace Rules

Configure submission rules, content policies, forbidden patterns, and category definitions.

Analytics

Marketplace-wide install trends, category health, top publishers, and search term analysis.

Audit Log

Immutable log of every admin action — who did what, when, and with what justification.

Issue Management

Triage and resolve reported policy violations, package complaints, and user disputes.

Rewards Management

Manually award points, grant badges, promote tiers, and run bulk reward campaigns for publishers.
