Reviews and Ratings

The reviews section provides social proof and qualitative feedback from users who have actually installed and used the package — gated behind a verified install check to maintain review integrity.

Review Eligibility

To leave a review, a user must have installed the package in their BizFirstGO tenant at least once. This requirement — the "verified install" gate — ensures that reviews come from genuine users rather than anonymous opinions or competitive manipulation.

ConditionCan Review?

Authenticated, never installed this packageNo — must install first
Authenticated, has installed this packageYes — can leave one review per package

Review Moderation

Reviews are visible immediately after submission — there is no pre-moderation queue. However, any user can flag a review for admin attention. Admins review flagged content and can remove reviews that violate the community standards (spam, abusive language, unrelated content).

Publishers cannot remove reviews themselves. Publisher responses can be added to any review — the response appears directly below the review text with a "Publisher Response" label.

Publisher tip: respond to reviews

Responding to both positive and negative reviews demonstrates active engagement. A thoughtful response to a critical review — acknowledging the issue and noting it is fixed in the latest version — is often more persuasive to prospective users than the negative review itself.

Rating Impact on Reputation

Only 5-star reviews contribute to publisher reputation points (+10 pts per 5-star review received). Reviews of 4 stars or fewer do not award reputation points — this prevents inflation of the reputation system by reviews that indicate the package needs improvement.
