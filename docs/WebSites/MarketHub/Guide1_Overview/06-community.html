MarketHub

Community

The MarketHub community connects package users and publishers through forums, Q&A, and announcements — building collective knowledge around the packages that power the ecosystem.

Forum Sections

General

Open discussion for the entire marketplace community. Platform news, tips, ecosystem discussions.

Package Forums

One dedicated forum per package. Publisher-moderated. Questions, bug reports, feature requests.

Announcements

Admin and publisher-only posting. Platform announcements, release notes, policy changes.

Q&A

Question-and-answer format with accepted answer mechanic. Upvotes surface the best answers.

Accepted Answer Mechanic

In Question threads, the thread author or the package publisher can mark one reply as the Accepted Answer. The accepted answer is pinned to the top of the reply list with a green checkmark indicator. The user whose reply was accepted earns +5 reputation points.

This mechanic incentivizes quality answers and makes it easy for later visitors to find the correct solution without reading the entire thread.

Moderation System

Community moderation uses a three-strike system. Publishers moderate their own package forums; admins moderate the General, Announcements, and Q&A sections.

StrikeTriggerResultDuration
Strike 1First policy violationFormal warning, post removedWarning permanent on record
Strike 2Second violation within 90 daysTemporary posting ban7 days

Community and Reputation

Community activity is the lowest-yield but most consistent source of reputation points. Publishers and users who are consistently helpful in Q&A forums accumulate +5 points per accepted answer, which adds up meaningfully over time — especially for active publishers in busy package forums.

Publisher tip: monitor your package forums

Package forum response time is measured and displayed on your publisher profile as a "Response Time" badge. Fast, accurate answers improve user trust and install conversion rates — and earn community reputation points.
