MarketHub Overview

The BizFirst marketplace — a three-app platform for discovering, publishing, and governing packages across the entire BizFirst ecosystem.

What is MarketHub?

MarketHub is the central marketplace for BizFirst packages. It connects publishers who build automation artifacts with the organizations and developers who want to use them. MarketHub handles discovery, reputation, certification, and community — while its sister system InstallHub handles the technical import pipeline.

Every package listed on MarketHub began life as an artifact in the BizFirst ecosystem: a Flow Studio process, an Atlas Form, an App Studio application, or a node library. Publishers export these artifacts through InstallHub and then list them on MarketHub for the world to discover.

Two systems, one ecosystem

MarketHub = discovery, reputation, community, and governance. InstallHub = the technical export/import pipeline. They are separate systems that work together: MarketHub's "Install" button launches the InstallHub import flow.

Three Applications

Admin Portal

Governance and management. Admins review submitted packages, moderate reviews, configure marketplace rules, run analytics, manage the audit log, and operate the rewards system.

Publisher Workspace

Where package authors live. Publishers manage their packages (draft to published), track installs and earnings, pursue certifications, manage API keys, and engage with their user community.

Public Discovery

Open to everyone. Users search and browse packages, read reviews, evaluate publishers, and install packages into their BizFirst tenant — all from a single discovery interface.

Publisher Reputation at a Glance

MarketHub uses a points-based reputation system that rewards publishers for real marketplace impact: installs, positive reviews, earned certifications, and community participation. Reputation accumulates into one of four tiers.

TierPoints RequiredKey Benefit
Bronze0 – 999Standard marketplace listing
Silver1,000 – 4,999Priority support queue
Gold5,000 – 19,999Featured placement eligibility
Platinum20,000+Priority search ranking boost
No downgrade — ever

Once a publisher reaches a tier threshold, they stay at that tier. Reputation decay (older installs and reviews weigh less) affects point totals but never causes a tier demotion.

Package Types

Flow Studio

Workflow processes and templates — automation sequences ready to import and run or customize.

Atlas Forms

Pre-built form definitions with validation, layout, and data-binding — import and embed immediately.

App Studio

Full application packages — UI layouts, widgets, navigation, and action systems bundled together.

Node Libraries

Custom execution node collections extending the Flow Studio node palette for specialized automation.
