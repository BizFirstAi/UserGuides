Point Values

Each event type awards a specific number of reputation points, calibrated so that broad marketplace impact (installs and reviews at scale) produces comparable tier progression to deep quality investment (certifications and community leadership).

Point Value Table

EventPointsRationale
Package install (per unique tenant)+1 ptVolume-based signal — scales naturally with package reach
5-star review received+10 pts10x install weight — a single glowing review signals deliberate quality approval
Certified certification earned+100 ptsLargest single event — reflects sustained quality investment and human review
Q&A accepted answer+5 ptsMid-tier reward for active, helpful community participation
Admin awardVariesDiscretionary — used for special contributions, corrections, or incentives

Progression Equivalencies

Understanding what combinations of events produce tier thresholds helps publishers plan their growth strategy:

To Reach Silver (1,000 pts)Example Path
Via installs only1,000 unique installs across all packages
Via reviews only100 five-star reviews
Via certification + installs1 Certified cert (100 pts) + 900 installs
Via mixed activity500 installs + 30 reviews + 2 certifications + 100 accepted answers = 500 + 300 + 200 + 500 = 1,500 pts (Gold approach)
To Reach Gold (5,000 pts)Example Path
Via installs only5,000 unique installs
Via reviews only500 five-star reviews
Via certifications + installs10 certs (1,000 pts) + 4,000 installs
Via full mixed activity2,000 installs + 100 reviews + 5 certs + 200 answers = 2,000 + 1,000 + 500 + 1,000 = 4,500 pts (near Gold)
To Reach Platinum (20,000 pts)Example Path
Via installs only20,000 unique installs
Via certifications + installs50 certs (5,000 pts) + 15,000 installs
Via full mixed (large publisher)10,000 installs + 300 reviews + 20 certs + 500 answers = 10,000 + 3,000 + 2,000 + 2,500 = 17,500 pts (close to Platinum with some admin awards)

Design Philosophy Behind the Values

Volume and Quality Both Matter

The 1pt/10pt/100pt ratio means a publisher can reach Silver through pure install volume (1,000 installs) or through fewer, higher-quality signals. The system rewards breadth and depth equally.

Certification is the Fast Track

At 100 points each, certifications are the highest-ROI activity. A publisher who invests in quality documentation and test coverage and earns multiple certifications will tier-progress significantly faster than pure install accumulation.

Community Scales Credibly

Accepted answers at +5pts provide meaningful accumulation for active community contributors without creating a community-only shortcut to top tiers. Reaching Gold through accepted answers alone requires 1,000 accepted answers — genuine expertise at scale.

No Artificial Inflation

The once-per-tenant rule on installs and the 5-star-only rule on reviews prevent simple inflation tactics. Publishers cannot accumulate points through repeated actions on the same entities.

Points are cumulative across all packages

A publisher's total reputation score is the sum of points earned across all their packages. A publisher with 10 packages each generating 500 install points reaches the same tier as one with a single package generating 5,000 installs.
