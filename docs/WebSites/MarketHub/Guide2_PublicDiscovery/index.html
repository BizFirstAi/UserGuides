MarketHub

Public Discovery

The public MarketHub site is the discovery layer of the BizFirstGO ecosystem — where users search, browse, evaluate, and ultimately install packages that extend their platform capabilities.

What Public Discovery Covers

The public discovery site is open to everyone, including anonymous visitors. It provides full read access to the marketplace — package listings, publisher profiles, community forums, and the publisher leaderboard — without requiring any account. Installation requires a logged-in BizFirstGO account.

Homepage

Featured packages, trending, new arrivals, and category shortcuts. The curated entry point to the marketplace.

Search

Full-text search with live filter controls for type, trust level, price, and minimum rating.

Categories

Browse by package type and domain — curated collections for common use cases.

Leaderboard

Top publishers ranked by reputation score — updated daily. A quality signal for the ecosystem.

Certifications

Browse certified-only packages — filter by certification level to find reviewed, trusted packages.

Publisher Profiles

Full publisher identity pages with tier badge, portfolio, community activity, and reputation score.

Search Ranking Formula

Search results are not simply sorted by install count. MarketHub uses a weighted ranking formula that balances multiple quality signals to surface the most relevant and trustworthy results.

SignalWeightWhat It Measures
Install count30%Real-world adoption — how many tenants have installed this package
Average rating25%User satisfaction — calculated from all submitted reviews
Recency20%How recently the package was updated — rewards active maintenance
Publisher reputation tier15%Platinum and Gold publishers get a ranking boost
Certification level10%Official > Certified > Community in tiebreaker
