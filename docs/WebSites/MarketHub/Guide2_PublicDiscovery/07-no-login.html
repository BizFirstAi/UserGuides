Anonymous Browsing

MarketHub is designed to be fully open for exploration — anonymous visitors can browse, search, and evaluate packages without creating an account. An account is only required to install or participate.

What Anonymous Users Can Access

The public catalog is completely open. No account, no tracking wall, no hidden content. The philosophy is that friction before evaluation reduces ecosystem adoption — so MarketHub shows everything to everyone.

The Install Gate

When an anonymous user clicks the Install button on any package, MarketHub redirects them to the BizFirstGO authentication page. After successful login, they are returned to the package detail page with the install flow already initiated — the original intent is preserved across the login redirect.

Intent preservation after login

MarketHub stores the intended package install as a session parameter before redirecting to login. After authentication, the user lands back on the package detail page and the install flow begins automatically — they do not need to find the package again.

Privacy and Tracking

Anonymous visits to MarketHub are logged for aggregate analytics only — page views, search terms, and category clicks contribute to marketplace trend data but are not linked to individual visitors. No cookies or tracking pixels are set on anonymous sessions beyond session management.
