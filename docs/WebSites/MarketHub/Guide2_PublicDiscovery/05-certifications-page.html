Certifications Page

The certifications discovery page lets users browse the marketplace filtered specifically to certified packages — surfacing the most thoroughly reviewed and trusted packages in each category.

Purpose of the Certifications Page

Enterprise users often want to restrict their evaluation to packages that have passed human review rather than browsing the full catalog. The certifications page pre-applies the trust level filter to show only Certified and Official packages, organized by certification level and then by category.

Certifications Page Sections

Official Packages

BizFirstGO-authored and partner packages at the highest trust level. Displayed first on the page.

Certified Packages

Community-published packages that have passed the human review process. The main body of the page.

By Category

Filter certified packages to a specific domain — Workflows, Forms, Apps, or Node Libraries.

Recently Certified

Packages that earned certification in the last 30 days — new additions to the trusted catalog.

Recommended for enterprise deployments

Organizations with governance requirements for package sourcing should restrict their installations to Certified and Official packages. Use the trust level filter in search or bookmark the certifications page as the starting point for package evaluation.
