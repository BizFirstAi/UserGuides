Publisher Dashboard

The publisher workspace is where package authors manage the complete lifecycle of their marketplace presence — packages, certifications, reputation, community, earnings, API keys, and issues, all in one dashboard.

Dashboard Overview Panel

The publisher dashboard home page shows a key metrics panel at the top — a quick view of the publisher's current standing.

MetricDescription
Published packagesCount of packages in Published status
Total installsCumulative installs across all published packages
Installs (30 days)Installs in the last 30 days — trend indicator with sparkline chart
Average ratingWeighted average rating across all packages
Reputation scoreCurrent total weighted reputation points with tier badge
Open issuesCount of unresolved issues against your packages
Pending submissionsPackages currently Under Review awaiting admin decision

Eight Dashboard Sections

My Packages

All packages across all statuses. Submit new versions, edit listings, withdraw packages.

My Certifications

Earned certifications, in-progress applications, and available cert criteria to pursue.

Reputation

Points breakdown, tier progress bar, recent point events, and badge collection.

Community

Package forum management, Q&A moderation, post history, and follower count.

Earnings

Revenue from paid packages, payout history, pending balance, and bank details.

API Keys

Generate and revoke API keys for programmatic CI/CD publishing workflows.

My Issues

Bug reports against your packages from users. Respond, track, and close.
