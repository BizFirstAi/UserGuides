Reputation

The reputation page in the publisher workspace shows the complete breakdown of earned points — by source, by package, and by date — along with tier progression, the badge collection, and recent point events.

Reputation Overview

Displayed ValueDescription
Weighted ScoreTotal score after decay is applied — this is the leaderboard ranking value
Lifetime PointsAll-time accumulated points without decay — this determines tier threshold
Current TierBronze / Silver / Gold / Platinum with tier progress bar to next tier
Leaderboard RankCurrent position in the publisher leaderboard with daily delta
Points This MonthPoints earned since the start of the current calendar month

Tier Progress Bar

A visual progress bar shows how close the publisher is to the next tier threshold based on lifetime points. For Platinum publishers (the highest tier), the bar instead shows total lifetime points with no "next tier" to reach.

Point Events Feed

The point events feed shows every individual point-earning event in reverse chronological order — the most detailed view of where reputation comes from:

Badge Collection

All earned badges are displayed in a collection view. Badges not yet earned are shown greyed out with the criteria needed to unlock them — giving publishers a motivating view of what to work toward next.

Publisher tip: focus on install quality over quantity

A single install earns 1 point. A 5-star review earns 10 points. And a Certified certification earns 100 points. The fastest path to tier progression is not maximizing installs alone — it is pursuing Certified status on your most-installed packages and building a base of satisfied users who leave 5-star reviews.
