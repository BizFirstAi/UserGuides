MarketHub

Published Packages

The published packages section on the publisher profile shows the publisher's complete marketplace portfolio — all currently live packages with their trust level, install count, rating, and pricing.

Package Grid Layout

Packages are displayed in a responsive card grid — three columns on desktop, two on tablet, one on mobile. Each card is identical in structure to the package cards in search results, ensuring users have a familiar experience as they navigate from profile to package detail.

Package Card on Profile

Each package card on the profile grid shows the same fields as on search results:

  • Trust badge (Official / Certified / Community)
  • Package name — links to the package detail page
  • Short description (first 100 chars)
  • Average rating with review count
  • Install count
  • Pricing (Free or price)
  • "Installed" badge — visible to authenticated users who have installed this package

Filter Controls

Visitors can filter the package grid by type without leaving the publisher profile page:

  • All Types — show all published packages
  • Workflows — Flow Studio packages only
  • Forms — Atlas Form packages only
  • Apps — App Studio packages only
  • Node Libraries — Execution node packages only
Delisted and draft packages are not shown

The public profile only shows packages with Published status. Packages that are Draft, Submitted for review, or Delisted are hidden from the public profile — they are only visible to the publisher in their workspace.
