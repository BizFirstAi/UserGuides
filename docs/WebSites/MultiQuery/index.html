Multi-Query Engine

Turn a single JSON template into rich hierarchical data — as structured JSON or an interactive HTML report — with one API call.

What is Multi-Query?

The BizFirst Multi-Query Engine lets you define a tree of related SQL queries as a JSON template, store it in the database, and retrieve fully nested, parent–child structured results with a single HTTP request or a single expression directive — no application code changes required.

Think of it as a data assembly engine: you describe what you need (the template), and the engine figures out how to execute, join, and assemble the result tree.

Who is this guide for?

This is the User Guide — written for administrators, integration engineers, and developers who configure templates, call the API, and maintain the engine in production. For code-level integration details, see the Developer Guide.
