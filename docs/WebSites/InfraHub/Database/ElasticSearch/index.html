Elastic Search for BizFirst

Complete guide to Elastic Search integration — full-text search, analytics, aggregations, Query DSL, performance tuning, index lifecycle management, and production clustering.

What is Elastic Search in BizFirst? Elastic Search is the full-text search and analytics engine for BizFirst. Workflows index documents to enable fast search across workflows, agents, datasets, and application data. The platform provides schema management, multi-tenant index isolation, automatic sharding, aggregation pipelines, and monitoring. Developers interact through IElasticProvider.IndexAsync() and IElasticProvider.SearchAsync() interfaces.
