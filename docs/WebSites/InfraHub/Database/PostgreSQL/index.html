PostgreSQL Database Integration

Complete guide to PostgreSQL database integration for BizFirst — connection pooling, schema design, query optimization, replication, backup strategies, and production deployment.

What is PostgreSQL in BizFirst? PostgreSQL is the primary relational database for BizFirst application data, tenant records, workflow state, and analytics. The platform provides connection pooling via PgBouncer, automatic credential management, multi-tenant schema isolation, backup automation, and point-in-time recovery. Developers interact through a single IPostgresProvider.ExecuteAsync() interface that handles pooling, tenant isolation, and credential resolution.
