FAQ

Can I implement my own provider instead of waiting?

Yes! See Bring Your Own Provider. Submit a PR and we'll review for inclusion.

How long does implementation take?

Typically 4-8 weeks depending on API complexity.

Do you accept sponsored implementations?

Yes! Contact partnerships@bizfirst.com for commercial arrangements.

What if my provider isn't on the roadmap?

Submit a request! If there's sufficient demand, it gets prioritized.

What if a provider shuts down?

We keep the dispatcher code but mark as deprecated. You can switch to another provider without code changes.
