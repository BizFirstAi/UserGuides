BizFirst Email System

Complete guide to multi-tenant, multi-provider email delivery with SMTP, AWS SES, and Gmail OAuth2. Production-ready architecture supporting custom providers.

What is BizFirst Email System? A plugin-based email delivery system supporting SMTP, AWS SES, and Gmail OAuth2. Each tenant configures their own email provider via named credentials stored in an encrypted vault. Dispatchers map credentials to providers and handle delivery. Extensible architecture allows adding new providers without modifying core code.

Architecture at a Glance

LayerComponentsResponsibility
Application Auth Service, Notification Service Calls IEmailService.SendAsync()
Core Orchestration EmailService, EmailDispatcherFactory Credential lookup, provider dispatch, result mapping
Providers SMTP, SES, Gmail dispatchers Type mapping, credential casting, service invocation
Integration Services ISmtpEmailService, ISesEmailService, GmailMessageService Actual email delivery (MailKit, AWS SDK, Google API)
Infrastructure Credential Vault, Encryption, Database Named credential resolution, AES-256-GCM encryption
