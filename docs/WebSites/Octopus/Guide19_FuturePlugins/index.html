Octopus

Future Plugins Roadmap

The Octopus plugin ecosystem is growing. This guide documents planned first-party plugins on the roadmap and the design principles every new plugin — first-party or community — must follow.

Roadmap Categories

Email Plugin

Read and send email via IMAP/SMTP, SendGrid, and Outlook. Agents can check inboxes, draft and send messages, and reply to conversations.

Details →

Calendar Plugin

Integrate with Google Calendar and Outlook Calendar — read events, create meetings, check availability, send invitations.

Details →

CRM Plugin

Read and write contacts, leads, opportunities, and activities in Salesforce and HubSpot. Agents can update deal stages and log calls.

Details →

ERP Plugin

Integrate with SAP and Microsoft Dynamics — purchase orders, invoices, inventory queries, vendor management.

Details →

Ticketing Plugin

Jira, ServiceNow, Zendesk — create, update, and query tickets. Agents can manage their own ticket queue or triage incoming issues.

Coming soon

Data Plugin

Query relational databases (SQL Server, PostgreSQL) and data warehouses (Snowflake, BigQuery) from agent tool calls with read-only safety guarantees.

Coming soon
